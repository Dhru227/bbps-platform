Configure Your Instance
Follow these steps in order
1
Clone the repository
Download the integrator to your machine.
git clone https://github.com/Dhru227/bbps-biller-integrator.git
cd bbps-biller-integrator
2
Create your PostgreSQL database
Create the database and schema for your bill data.
psql -U postgres -h localhost -c "CREATE DATABASE billerdb;"
psql -U postgres -h localhost -d billerdb -c "CREATE SCHEMA IF NOT EXISTS billerdb;"
cat src/main/resources/schema.sql | psql -U postgres -h localhost -d billerdb
3
Configure application.properties

4
Seed your bill data
Add sample bills so fetch requests return real data during the demo.
psql -U postgres -h localhost -d billerdb -c "
INSERT INTO billerdb.bill_details
  (customer_param_name, customer_param_type,
   customer_param_value, bill_amount,
   bill_date, due_date, bill_status, bill_number)
VALUES
  ('Consumer Number','Numeric','1001',1850.00,'2026-04-15','2026-05-01','UNPAID','DEMO-001'),
  ('Consumer Number','Numeric','1002',3200.00,'2026-04-15','2026-05-01','UNPAID','DEMO-002'),
  ('Consumer Number','Numeric','1003',750.00,'2026-04-15','2026-05-01','UNPAID','DEMO-003');"
5
Start the integrator
Run this from inside the cloned repository folder.
mvn spring-boot:run
ℹ️ Requires Java 17 and Maven — Download Java 17 · Download Maven
6
Verify it's running
Open a new terminal and run this to confirm your integrator is live.
curl -X POST http://localhost:8111/mock-biller/test/fetch \
  -H "Content-Type: application/json" \
  -d '{"Consumer Number": "1001"}'
Expected: bill_amount, due_date, status: SUCCESS
